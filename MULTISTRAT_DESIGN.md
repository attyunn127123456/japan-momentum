## 2. 各ポッドの概要

### 2.1 Momentum Pod（既存拡張）
- **シグナル**: 既存のクロスセクショナルモメンタム（optimize.pyのファクター群）
- **ロング**: top_n銘柄をスコア順にロング
- **ショート**: bottom_n銘柄をショート（信用取引）
- **リバランス**: 週次（月曜寄付き）
- **エグジット**: トレーリングストップ + 天井検知（既存compute_exit_signal）
- **Phase 1で実装**: ✅（既存コード活用）

### 2.2 Mean Reversion Pod（Phase 2で実装予定）
- **シグナル**: RSI < 30 + ボリンジャーバンド下限突破 + 出来高急増
- **ロング**: 短期過売り銘柄（3-5日保有）
- **ショート**: 短期過買い銘柄（RSI > 70）
- **リバランス**: 日次
- **エグジット**: 利益確定 +3% / 損切り -2%

### 2.3 Event Driven Pod（Phase 3で実装予定）
- **シグナル**: 
  - 決算サプライズ（EPS成長率 vs コンセンサス）
  - 信用倍率急変（買い残/売り残の変化率）
  - 自社株買い・増配発表
- **データソース**: J-Quants ファンダメンタルAPI
- **保有期間**: 1-4週間

### 2.4 Macro Overlay Pod
- **役割**: ポートフォリオ全体のリスク調整
- **シグナル**:
  - 日経平均100日MA割れ → 全体エクスポージャー縮小
  - VIX相当（日経VI）上昇 → ヘッジ増加
  - 為替（USD/JPY）急変 → 輸出入銘柄の調整
- **アクション**: 各ポッドの投資比率を動的に調整（0%〜150%）
- **Phase 1で実装**: ✅（既存market_regime_filter活用）

### 3.1 データ更新サイクル（日次）
1. **06:00 JST**: J-Quantsからデータ取得（前日約定分）
2. **06:05**: ファクター再計算 → シグナル生成
3. **06:10**: ペーパートレード更新（前日ポジションの時価評価）
4. **06:15**: リスクチェック（DD・VaR・エクスポージャー）
5. **06:20**: シグナル出力 → Discord通知
6. **08:45**: 寄付き前の最終確認（レジーム判定）
7. **09:00**: 寄付き（=シグナルに基づく仮想約定）

## 4. ペーパートレーディング仕様

### 4.1 取引コスト
| 項目 | 設定値 | 備考 |
|------|--------|------|
| スリッページ（片道） | 0.15% | 往復0.30% |
| 委託手数料 | 0% | SBI/楽天のゼロコース想定 |
| 信用金利（ロング） | 年2.80% | 制度信用 |
| 貸株料（ショート） | 年1.10% | 制度信用 |
| 逆日歩 | 0円 | 簡易モデル（Phase 2で可変化） |
| 最低取引単位 | 100株 | 単元株制限 |

### 4.2 約定シミュレーション
- **エントリー**: リバランス日の始値（AdjO）+ スリッページ
  - ロング: `entry_price = open_price * (1 + 0.0015)`
  - ショート: `entry_price = open_price * (1 - 0.0015)`
- **エグジット**: リバランス日の始値 or ストップ発動時の終値
  - ロング: `exit_price = open_price * (1 - 0.0015)`
  - ショート: `exit_price = open_price * (1 + 0.0015)`

### 4.3 信用取引シミュレーション
- **保証金率**: 30%（最低委託保証金率）
- **追証ライン**: 保証金維持率 25%未満
- **レバレッジ**: 最大3.3倍（保証金率30%の逆数）
- **実運用時のレバレッジ目標**: 1.0〜1.5倍

## 6. リスク管理エンジン（portfolio_engine.py）

### 6.1 リスク予算管理
- 各ポッドに独立したDDリミットを設定
- DDリミット到達時: ポッドのポジションを50%縮小
- DDリミット×1.5到達時: ポッド停止（全ポジション清算）
- ポートフォリオ全体DDリミット到達時: 全ポッド停止

### 6.2 VaR計算
- **手法**: Historical Simulation (過去252日)
- **信頼水準**: 95%, 99%
- **計算頻度**: 日次

### 6.3 ポッド間相関監視
- 30日ローリング相関を日次計算
- 相関 > 0.7 のペアが出現: アラート発出
- 分散効果が低下した場合: 配分比率の自動調整を提案

### 6.4 ポジションサイズ制限
| 制限 | 値 | 備考 |
|------|-----|------|
| 1銘柄上限（対NAV） | 10% | 集中リスク回避 |
| 1セクター上限（対NAV） | 25% | セクター分散 |
| グロスエクスポージャー上限 | 200% | レバレッジ制限 |
| ネットエクスポージャー範囲 | -50%〜+150% | 方向性リスク制限 |
