## 仮説メモ (2026-03-05 あつし)

### 「本物のモメンタム」の定義

単に今日上がってるじゃなくて：
1. **じわじわ伸び続けている**（R²の高いなだらかな上昇トレンド）
2. **下落局面での反発が速くて強い**

### 下落反発指標のアイデア
日経全体が下がった日（例: 日経が-1%以下の日）を「下落日」と定義して：

```python
# 下落日の定義
down_days = nikkei_ret[nikkei_ret < -0.01].index  # 日経-1%以上の日

# 銘柄のその日のリターン
stock_on_down_days = daily_rets[down_days]

# 指標1: 下落日の平均リターン（高いほど強い）
resilience = stock_on_down_days.mean()

# 指標2: 下落日の翌日リターン（反発速度）
next_day_rets = daily_rets.shift(-1)[down_days]
rebound_speed = next_day_rets.mean()

# 指標3: 下落局面での最大DD（小さいほど強い）
```

### ポジション管理アイデア
- 「強い銘柄に資産を寄せ続ける」
- 単純均等配分ではなく、スコアに比例した加重配分
- 上位銘柄でもモメンタムが弱まってきたら比重を下げる

### 実装優先度
1. R²（トレンド滑らかさ）→ 材料系除外
2. resilience（下落日相対強度）→ 本物モメンタム識別
3. rebound_speed（反発速度）→ 同上
4. スコア比例ウェイト → ポジション管理

## 仮説修正 (2026-03-05)

### 急騰は問題じゃない
- 急騰後も伸び続けるなら買い
- 問題は「急騰後に失速・反落するもの」

### 経験則ベースのエントリー/エグジット

**買いシグナル：新高値ブレイク**
```python
# 過去N日の高値を更新したタイミング
new_high = price > prices.rolling(60).max().shift(1)
```

**売りシグナル：大出来高の陰線**
```python
# 出来高が平均の2倍以上 かつ 陰線（終値 < 始値）
big_vol = volume > volume.rolling(25).mean() * 2
bearish_candle = close < open_price
exit_signal = big_vol & bearish_candle
# → 機関が売り始めたサイン
```

### 解釈
- 新高値ブレイク = 売り圧力を吸収して上抜け → 需給が強い証拠
- 大出来高陰線 = 機関の利食い・売り転換のサイン → 需給が崩れる前兆

### スコアへの組み込み案
1. 新高値ブレイクからの経過日数 → 短いほどエントリー好機
2. 直近N日で大出来高陰線があったか → あればペナルティ or 除外
